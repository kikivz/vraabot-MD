## FOR TERMUX/UBUNTU/SSH USER

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
git clone https://github.com/kikiivz/kikibot-Md
cd kikibot-Md
npm install
```

## IF ERROR INSTALL MODULE FOR TERMUX

```bash
pkg install yarn
yarn
```
